# 🧬 Protein Folding Agent System

A multi-agent AI system for protein structure prediction and optimization using large language models and bioinformatics workflows.

## 🚀 Overview

This project implements a **multi-agent collaborative framework** for predicting and refining protein 3D structures from amino acid sequences.

The system integrates:
- LLM-based reasoning (GPT / Claude / Gemini)
- Bioinformatics preprocessing
- Iterative structural optimization
- Energy-based evaluation

## 🧠 Architecture

The system is composed of multiple specialized agents:

- **Data Agent**
  - Sequence cleaning
  - Feature extraction
  - Tokenization

- **Folding Agent**
  - Initial structure prediction
  - LLM-assisted reasoning on folding patterns

- **Evaluation Agent**
  - Structural validation
  - Energy scoring
  - Stability estimation

- **Optimization Agent**
  - Iterative refinement
  - Multi-step reasoning
  - Convergence control
